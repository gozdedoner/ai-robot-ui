# AI Robot Chat 🤖

A simple AI chatbot interface built with React, TypeScript, and OpenAI API.

## 🚀 Features
- Real-time chat UI
- OpenAI API integration
- TypeScript support
- Custom chat bubbles

## 🛠️ Tech Stack
- React
- TypeScript
- Vite
- OpenAI API
